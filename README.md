# MontyHallGame

## MontyHallGameServer (port:`1214`)

- 使用[Ktor](https://ktor.io/)作为服务器框架，主要使用`AtomicInteger`存储数据。

## MontyHallGameWebfront (port:`1215`)

- 使用[Vue3](https://cn.vuejs.org/)作为前端框架。

> [!TIP]
> 前端代码大量使用了[Gemini 2.5 Pro](https://docs.cloud.google.com/vertex-ai/generative-ai/docs/models/gemini/2-5-pro?hl=zh-cn)。
